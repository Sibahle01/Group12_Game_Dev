# Bug Squash 🐛

A 2D top-down action game built with Godot 4 by Group 12.

## Game Concept
You are an IT technician defending servers from bugs in a server room. 
Kill all the bugs before they destroy your servers!

## How to Play
- **WASD** — Move
- **Space** — Attack (melee swing)
- **F** — Place Firewall (blocks bugs temporarily, 3 per level)
- **Coffee** — Pick up for a speed boost when it appears

## Features
- 3 levels with increasing difficulty
- Multiple bug types (roach, beetle, ant, mantis boss)
- Servers that bugs target and damage
- Firewall placement mechanic
- Coffee powerup for speed boost
- Boss battle on Level 3 with unique music
- Game over when all servers are destroyed

## Controls Summary
| Key | Action |
|-----|--------|
| WASD | Move |
| Space | Attack |
